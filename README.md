# Gravix Signup UI

A clean, responsive signup page built using HTML and CSS. 
This project focuses on layout structure, form validation, and modern UI styling without any JavaScript.

---

## Overview

Gravix is a two-panel registration interface:

- **Left Panel** – Branding section with logo and tagline 
- **Right Panel** – Signup form with floating labels and validation styles 

The design emphasizes clarity, responsiveness, and user-friendly form interactions.

[Live URL](https://mukul-work.github.io/responsive-signup-ui/)

---

## Features

- Responsive layout using Flexbox and CSS Grid 
- Floating label inputs
- Native HTML validation (required fields, email, phone pattern) 
- Visual feedback for valid and invalid input states 
- Custom reset stylesheet
- Modern color scheme with gradient accents 
- Google Fonts integration (Inter)

---

## Tech Stack

- HTML5 
- CSS3 
- Flexbox 
- CSS Grid 

No frameworks or JavaScript used.

---


---

## Form Validation

The form uses native HTML validation:

- `required` attribute for mandatory fields 
- Email input type validation 
- Phone number pattern validation 
- CSS-based visual state changes (`:focus`, `:valid`, `:invalid`) 

No client-side or server-side scripting is implemented.

---

## Responsive Design

Breakpoints:

- 1024px – Layout stacks vertically 
- 768px – Single-column form layout 
- 480px – Adjusted typography and spacing 

The interface adapts smoothly across desktop, tablet, and mobile screens.
